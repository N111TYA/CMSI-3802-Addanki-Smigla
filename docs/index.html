Forge

A statically typed language that compiles to JavaScript — built for clarity, precision, and craft.

View on GitHub Jump to Examples

The Story

Forge was built around a simple question: what if writing low-level logic felt as deliberate as metalwork? Most languages either enforce rigid ceremony or leave too much implied. Forge takes a different path — every function declares what it returns, every variable knows its type, and the compiler tells you exactly when something is wrong before your code ever runs.

Forge compiles to JavaScript through a full pipeline: a grammar defined in Ohm.js, a parser that builds a typed AST, a static analyzer that enforces thirteen safety checks, an optimizer that folds constants and eliminates dead branches, and a code generator that produces clean, readable JS. Control flow reads like intention — is instead of if, whenis for while loops, deliver to return a value, resolve to return nothing.

The language is intentionally small. Forge doesn't try to do everything — it tries to do its small set of things precisely and verifiably. It's a language for people who want to know exactly what their code is doing, and a compiler that makes that knowledge cheap.

Features

Strong Types

Nine primitive types: byte, char, snum, inum, lnum, fnum, dnum, hilo (boolean), and empty (void). Every function signature is explicit.

Expressive Control Flow

is/not (if/else), whenis (while), whento (for), execute/whenis (do-while), and hub (switch) — each keyword names the intent, not the mechanism.

Safety by Default

Sixteen static checks enforced at compile time — undeclared variables, bad assignments, wrong call signatures, type mismatches, and more. Errors surface before execution.

Bit-Level Precision

Native bit indexing (x{n}), bitwise word operators (bitand, bitor, shiftl, shiftr, bitnot), and array slicing (x[i:j]).

Custom Types

Enum-style custom declarations with optional literal payloads. hub switch dispatches on them cleanly, with a mandatory base default clause.

Compiles to JavaScript

A full optimizer + generator pipeline produces readable, idiomatic JS. No runtime, no bundler, no ceremony — just clean output you can read and trust.

Demonstrates: typed function signatures, is/not (if/else), recursion, deliver (valued return), resolve (void return).

// Hello World example — functions, strings, if/else

func greet(name: dnum): empty {
  resolve;
}

func max(a: inum, b: inum): inum {
  is (a > b) {
    deliver a;
  } not {
    deliver b;
  }
}

func factorial(n: inum): inum {
  is (n <= 1) {
    deliver 1;
  }
  deliver n * factorial(n - 1);
}

func abs(x: inum): inum {
  is (x < 0) {
    deliver -x;
  }
  deliver x;
}

// Math example — loops, arithmetic, exponentiation

func power(b: inum, e: inum): inum {
  let result = 1;
  whento (let i = 0; i < e; i++) {
    result *= b;
  }
  deliver result;
}

func sumTo(n: inum): inum {
  let total = 0;
  let i = 1;
  whenis (i <= n) {
    total += i;
    i += 1;
  }
  deliver total;

  whenis (b != 0) {
    let temp = b;
    b = a % b;
    a = temp;
  }
  deliver a;
}

      deliver Direction.East;

    :{3}: {
      deliver Direction.West;

    base {
      deliver -1;
    }
  }
}
